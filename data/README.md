# Data directory

This repository intentionally does not commit raw datasets.

Required official datasets:

1. IEEE-CIS Fraud Detection
   - Official source: Kaggle competition `ieee-fraud-detection`
   - Expected local path for full reproduction on Lust's machine:
     `D:/motif-entropy-fraud/ieee-fraud-detection`
   - Required files:
     - `train_transaction.csv`
     - `train_identity.csv`

2. CSE-CIC-IDS2018, optional diagnostic external cyber dataset
   - Official source: UNB/CIC + AWS Open Data Registry
   - Registry: https://registry.opendata.aws/cse-cic-ids2018/
   - Do not use label-derived attack names as detector hierarchy fields.

The GitHub-ready curated result artifacts are in `results/`, not `outputs/`.
`outputs/` is ignored because it is generated by reproduction scripts and contains smoke/test runs.
